藍莓(英文：Blueberry)

藍莓起源

藍莓原產於北美洲與東亞，是地球上最古老的植物之一，分佈於北美、南美、亞洲與歐洲。早在數百年前，美國當地土著印第安人已在食用藍莓，他們食用藍莓方式有許多種，包括將其乾燥後磨成粉，或做為肉類的調味料，讓其可長期保存食用，藍莓不僅可用於料理，藍莓營養極為豐富，當地婦女產後也會將它當為補品來補身子，藍莓本身所擁有的維生素C和維生素A，遠遠多於其他水果。顏色：深藍、暗紫、黑，大小不一。食用方式：鮮食、或製作成果凍、果醬、沙拉、鬆餅等等，

藍莓品種

越桔酸類植物(Vacinnium)是所有藍莓的總稱，包含450多種不同的品種。藍莓於世界各地皆有自然生長，也有不同的名字。於商業上，主要生產藍莓有三大類： Corymbosum(High-bush 或 Tall blueberry)，

高叢藍莓，北方高灌木，樹高5-15英呎(約152-457公分)，不過一般會將其修剪至6英呎(約182公分)左右，方便管理採收，此品種屬高需冷品種，適合冬季生長，產期約6-10月，主要產地：美國、加拿大、歐洲、智利、南非、紐西蘭、澳洲，智利所產藍莓是南半球最大出產國，也是北半球最大藍莓出口商，智利產期約10-3月

Angustifolium(Low-bush)，

矮叢藍莓，矮灌木，也被稱作【野生藍莓】，樹高僅1-2英呎(約30-60公分)，雖然如此，其生產的果相當豐富，此品種非常耐寒，適合北美的極地區栽種。從北極地區、明尼蘇達州、紐約的山上及新漢普郡，所產的甜藍莓屬此品種；而新英格蘭及整個西部地區所種的酸藍莓，則屬於myrtillus品種

Ashei(Rabbiteye)

南方兔眼，於美國南部相當普遍，此品種因其花萼和兔子的眼睛相似，故稱爲兔眼，於墨西哥最為著名。

藍莓產期

藍莓目前台灣可全年供應。北半球，以北美洲的為主，供應區域從佛羅里達(南美洲)到太平洋的西北部。而南半球以智利﹐阿根廷﹐澳洲和紐西蘭為主；美國於1999年開始，每年度的7月份都會舉辦藍莓節，而7/4是此節日的最高峰﹐也是美國的國慶日。
